Catnip

Priority based time management app

Project Description

Catnip is a time management app which helps users prioritise their tasks to reduce stress. Catnip was designed, tested and developed by our team using Python and Kivy.

Catnip is different from other calendar apps as it not only lists all the tasks for the user, it tries to guide the user into making a choice in what to prioritise.

Research

We researched behavioural change to learn how to help our users prioritise their time. We learned to involve users in planning their time to instil a sense of ownership.

We chose to use Python due to familiarity. For the user interface, we chose Kivy as it is a library written in Python and has cross-platform development capabilities.

Ideate

We held brainstorm sessions to generate ideas for user interfaces. We gave post-it notes to participants to sketch their ideas before we discussed them to not constrain their ideas.

Prototype

For the initial prototypes of the UI, we sketched out the wireframes to conduct user testing. After soliciting feedback from focus groups, we narrowed down the number of prototypes and used proto.io to build a more comprehensive prototype.

We chose to use a grid based layout for the main view. The grid represents the whole day, which forces the user to choose what he wants to include as there is a clear limit.

Development

We split the app into 4 main sections: priority view, list view, calendar view and the backend.

The backend was written in Python which consists of API calls to pull data from Canvas and CAESAR.

The data was then parsed for the desired information before being stored in JSAON files for persistent data storage.

Team

Allison Lu, Janus Kiong, Leslie Zhang, Lexy Praeger

Tools

Python, Kivy, JSON, Proto.io
