Trashifai

Smart bin that sorts waste using AI

Project Description

Trashifai is a smart bin that sorts waste by using AI to identify the waste and sort into recyclable/non-recyclables for easy waste management. Trashifai was conceptualized and built during Wildhacks 2016, a hackathon held at Northwestern University.

Ideate

We thought of using images to train an AI to identify waste because we wanted to use Clarifai, a platform for image recognition. We chose to do a waste sorting bin as it combines both hardware and software while solving a huge problem of wrongly sorted waste.

Hardware

We used an Arduino as the controller of the bin and a laptop for its camera to take a picture of the waste and to run the API call for the AI to identify the object. 2 ultrasound sensors were used to detect the remaining capacity of the bin.

Software

We trained the AI using thousands of images that split the objects into two groups, recyclables and non-recyclables. We used Python to run the API call to identify the image of the waste which then sends a signal to the Arduino to turn the servo to dispose the waste in the correct section. We used Twillo to send SMS to the owner when the ultrasound sensors detect the bins are full.
